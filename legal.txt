What Nx86 is
------------
Nx86 is an independent, clean-room Nintendo Switch emulator and AArch64-to-
x86_64 binary translation system. It is a tool. It operates only on content that
the user supplies.


What Nx86 does not contain or distribute
----------------------------------------
This repository and any official Nx86 build ship NONE of the following, and Nx86
must never request, bundle, embed, or download them:

  - copyrighted game dumps, ROMs, or game assets
  - proprietary console firmware or system software

  - proprietary SDK code or extracted runtime libraries
  - shared save files or other personal user data
  - copyrighted binary blobs of any kind, including inside shared profiles

Nx86 provides no keys, no firmware, and no means of obtaining them. Acquiring
the content needed to run a title is entirely the user's responsibility, and
that content must be legally obtained by the user.
